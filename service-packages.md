# QA Automation Service Packages

These packages are designed for small teams that need useful automation quickly without hiring a full-time QA automation engineer.

The offer is backed by a portfolio framework using Java, Playwright, REST Assured, TestNG, Allure, GitHub Actions, and Docker.

---

## Package 1: Starter QA Automation Pack - $350 CAD

Best for a small team that wants a first automated smoke suite before release.

### Includes

- 3-5 Playwright UI smoke tests
- 1-2 API smoke tests if endpoints are available
- Basic TestNG execution setup
- Simple bug report with screenshots if defects are found
- Test execution proof
- Clean README with setup and run commands
- GitHub repository handoff or ZIP handoff

### Suggested flows

- Login
- Registration
- Search
- Contact form
- Cart or checkout smoke flow

### Delivery time

3-5 business days

### Best for

- First-time automation setup
- Small web apps
- Startup MVPs
- E-commerce landing or checkout smoke coverage

## Scope Control

These packages are for smoke and regression automation, not full enterprise transformation.

Out of scope unless agreed separately:

- Full performance testing
- Security testing
- Large-scale test data engineering
- Complex multi-environment infrastructure
- Full database automation framework
- Mobile app automation
- Production monitoring

Scope control prevents the classic freelancing disaster where a $350 job slowly mutates into rebuilding a company’s engineering department. Nature is cruel.
