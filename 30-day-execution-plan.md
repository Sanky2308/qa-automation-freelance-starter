# 30-Day Execution Plan to Reach $1,000 CAD/month

## Week 1: Build Proof

### Goal
Create one polished QA automation portfolio project.

### Tasks
- Finalize GitHub framework
- Add README
- Add architecture diagram
- Add Allure screenshots
- Add GitHub Actions screenshot
- Add Docker execution screenshot
- Add sample bug report
- Pin repo on GitHub profile

### Success Criteria
Your repo should look good enough that a client can trust you without a long explanation.

---

## Week 2: Create Sales Assets

### Goal
Make profiles and service pages.

### Tasks
- Update LinkedIn headline
- Update LinkedIn About section
- Create Upwork profile
- Create Fiverr gig
- Prepare 3 Upwork proposal templates
- Prepare 5 LinkedIn outreach messages
- Create a simple portfolio post

### Success Criteria
You can apply/send messages daily without rewriting from zero every time.

---

## Week 3: Outreach Sprint

### Goal
Get conversations.

### Daily Targets
- 5 Upwork proposals
- 10 LinkedIn messages
- 5 agency emails

### Weekly Target
100 total outreach attempts.

### Success Criteria
At least 5 replies and 1 serious discussion.

---

## Week 4: Close First Client

### Goal
Sell the $350 starter package.

### Offer
'I can automate 5 critical flows and provide a test report for $350 CAD.'

### Tasks
- Offer fixed scope
- Avoid hourly confusion
- Deliver quickly
- Ask for testimonial
- Offer monthly maintenance

### Success Criteria
One paid project or one signed trial project.

---

## Daily Work Routine

| Time | Task |
|---|---|
| 30 min | Search jobs/leads |
| 45 min | Send proposals/messages |
| 30 min | Improve GitHub/profile |
| 30 min | Practice/demo automation |
| 15 min | Update tracker |

## Weekly Review

Every Sunday:
- Count messages sent
- Count replies
- Count calls booked
- Count proposals won
- Improve weak message
- Update screenshots and profile proof

## Main Rule

Do not keep preparing forever. Preparation without outreach is just procrastination wearing a tie.
