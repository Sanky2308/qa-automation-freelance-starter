## How to Use This

1. Polish the technical QA framework in `ecommerce-sdet-framework`.
2. Copy `portfolio-readme-template.md` into that technical repo if needed.
3. Use `profile-and-gig-copy.md` to update Upwork, Fiverr, and LinkedIn.
4. Send outreach messages daily using the supplied templates.
5. Track every lead in `qa-freelance-tracker.csv`.
6. Convert the first small project into recurring monthly QA maintenance.
